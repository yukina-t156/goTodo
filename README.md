# goTodo

TODOリストを作りたい！
- まずは基本から
- できたら拡張したい
  - チェックをつける
  - 優先度を設定
  - タスクにかける時間などをいれたい
  - ポモドーロタイマーをつけたい
  - タスクを25minごとに分けてポモドーロタイマーと連動させたい
  - どのタスクを何回分どの順番でやるかのタイムラインを作りたい
  - 完了済みのものと未完のものを分けて表示したい
