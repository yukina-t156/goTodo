# goTodo

TODOリストを作りたい！
- まずは基本から
- できたら拡張したい
  - タスクにかける時間などをいれたい
  - ポモドーロタイマーをつけたい
  - タスクを25minごとに分けてポモドーロタイマーと連動させたい
  - どのタスクを何回分どの順番でやるかのタイムラインを作りたい
